# 潜望量化 V1.0 产品需求文档

## 一、产品概述

### 1.1 产品定位

**潜望量化**是一款面向个人投资者的量化策略回测平台，提供策略管理、股票池管理、策略回测等核心功能，帮助用户快速验证量化交易策略的有效性。

### 1.2 目标用户

- 个人量化投资者
- 金融数据分析爱好者
- 量化策略开发者

### 1.3 产品愿景

让量化投资更简单，帮助每一位投资者找到适合自己的交易策略。

### 1.4 版本信息

| 属性 | 内容 |
|------|------|
| 版本号 | V1.0.1 |
| 发布日期 | 2026年7月 |
| 状态 | 正式版 |

## 五、数据库设计

### 5.1 用户表 (user)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 自增主键 |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(255) | NOT NULL | 密码(MD5) |
| email | VARCHAR(100) | NULL | 邮箱 |
| phone | VARCHAR(20) | NULL | 手机号 |
| status | TINYINT(1) | DEFAULT 1 | 状态: 0禁用 1启用 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 5.2 股票基本信息表 (stock_basic_info)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| ts_code | VARCHAR(20) | PRIMARY KEY | 股票代码 |
| name | VARCHAR(50) | NULL | 股票名称 |
| area | VARCHAR(20) | NULL | 地域 |
| industry | VARCHAR(50) | NULL | 所属行业 |
| market | VARCHAR(20) | NULL | 市场类别 |
| list_date | DATE | NULL | 上市日期 |
| list_status | CHAR(1) | NULL | 上市状态 |
| fullname | VARCHAR(100) | NULL | 全称 |
| is_hs | CHAR(1) | NULL | 是否沪深港通 |

### 5.3 策略类型表 (strategy_type)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 自增主键 |
| code | VARCHAR(50) | NOT NULL, UNIQUE | 策略编码 |
| name | VARCHAR(100) | NOT NULL | 策略名称 |
| description | TEXT | NULL | 策略描述 |
| category | VARCHAR(50) | NULL | 策略分类 |
| params | JSON | NULL | 默认参数配置 |
| is_active | TINYINT(1) | DEFAULT 1 | 是否启用 |

### 5.4 回溯测试结果表 (backtest_result)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 自增主键 |
| ts_code | VARCHAR(20) | NOT NULL | 股票代码 |
| stock_name | VARCHAR(50) | NULL | 股票名称 |
| strategy_type | VARCHAR(50) | NOT NULL | 策略类型 |
| strategy_params | JSON | NULL | 策略参数 |
| initial_capital | DECIMAL(15,2) | NOT NULL | 初始资金 |
| final_capital | DECIMAL(15,2) | NOT NULL | 最终资金 |
| total_return | DECIMAL(10,4) | NULL | 总收益率% |
| annualized_return | DECIMAL(10,4) | NULL | 年化收益率% |
| max_drawdown | DECIMAL(10,4) | NULL | 最大回撤% |
| sharpe_ratio | DECIMAL(10,4) | NULL | 夏普比率 |
| win_rate | DECIMAL(10,4) | NULL | 胜率% |
| total_trades | INT | NULL | 总交易次数 |
| test_date | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 测试时间 |

### 5.5 交易记录表 (backtest_trade)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|

| result_id | INT | NOT NULL | 关联结果ID |
| trade_date | DATE | NOT NULL | 交易日期 |
| action | VARCHAR(10) | NOT NULL | 交易方向 |
| price | DECIMAL(10,4) | NOT NULL | 交易价格 |
| quantity | INT | NOT NULL | 交易数量 |

### 5.6 资金曲线表 (backtest_equity_curve)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 自增主键 |
| result_id | INT | NOT NULL | 关联结果ID |
| trade_date | DATE | NOT NULL | 交易日期 |
| equity | DECIMAL(15,2) | NOT NULL | 账户总权益 |
| cash | DECIMAL(15,2) | NULL | 现金 |
| position_value | DECIMAL(15,2) | NULL | 持仓市值 |

## 六、API接口设计
